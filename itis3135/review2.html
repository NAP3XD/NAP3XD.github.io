Checklist for Each Page

  • Your submission leads right to the page to be reviewed (or if instructed the primary page with links to the page, or both)
  • There are no spaces or upper-cases in the file/folder names, including scripts, images, etc.
  • Design
    • Page has sufficient contrast/font sizing so as to be easy to read
    • Page has site colors and fonts using the standard.css file, unless this assignment requires a unique/standalone/embedded one
    • CRAP
      • Contrast: NA
      • Repetition: NA
      • Alignment: NA
      • Proximity: NA
  • Page has within it:
    • Header
    • Main
    • Footer
    • (optional) nav for main menu between header and main
  • Header has site/brand header with text in an h1
    • If this is a homework page it's likely something like Student Name Course ID Course Name
    • If this is a "branded page" it should be the brand/company
    • DO NOT INCLUDE THE PAGE NAME IN THE HEADER
    • Understand the difference between head, header, and heading
  • Main starts with name of page as an h2 and does not include name of site/brand
  • Page has brand tagline somewhere (make up a slogan and include it on every page in a consistent location)
  • Page has footer with:
    • Menu for user's pages
    • Page has HTML validation button/link and validates
    • Page has CSS validation button/link and validates
  • Page has specific requirements for this assignment (make a checklist from that assignment and make sure you've covered everything there). If there is a conflict between what you see here and in the instructions, ask on the class forums about it.
